# Revised Simplex Method

# Prerequisites
* Python 3.10 or newer
* Pip

# How to run locally
1. Clone the repository
2. Go to the root of our project:  
   `cd Simplex`
3. Install dependencies:  
   `python -m pip install -r requirements.txt`
4. Run program:  
   `python main.py`

# Run tests
`python -m unittest tests\test.py`
